## 1. はじめに

### ポモドーロテクニックとは？

ポモドーロテクニックは、1980年代にイタリアの Francesco Cirillo（フランチェスコ・シリロ）によって考案された時間管理術です。名前の由来は、彼が学生時代に使っていたトマト（イタリア語で pomodoro）型のキッチンタイマーから来ています。

基本的な考え方はとてもシンプルです。

1. **25分間** 集中して作業する（= 1ポモドーロ）
2. **5分間** の短い休憩を取る
3. これを繰り返し、**4ポモドーロ完了後** に **15〜30分間** の長い休憩を取る

### なぜ効果があるのか？

#### 「短い集中」が脳に優しい

人間の集中力には限界があり、長時間ぶっ通しで作業しても効率は下がる一方です。25分という短い区切りにすることで、「あと少しだけ頑張ろう」と思える適度なプレッシャーが生まれ、高い集中力を維持しやすくなります。

#### 「始めるハードル」が下がる

大きなタスクに取りかかるとき、「まず25分だけやってみよう」と思えば気軽に始められます。実際にやり始めると集中状態に入りやすくなる（作業興奮と呼ばれる現象）ため、最初の一歩を踏み出す助けになります。

#### 休憩が集中を持続させる

短い休憩は単なるサボりではなく、脳をリフレッシュするための大切な時間です。こまめに休憩を挟むことで、1日を通して安定したパフォーマンスを保つことができます。長い休憩では、軽い運動や水分補給など、体のケアをするとさらに効果的です。

#### 作業を「見える化」できる

「今日は何ポモドーロ完了した」と記録することで、自分の作業量が目に見える形になります。感覚ではなく数字で振り返れるため、計画の見積もり精度が上がり、日々の改善につながります。

### こんな方におすすめ

- 作業に集中できず、つい他のことに気を取られてしまう方
- 大きなタスクを前にすると、なかなか手をつけられない方
- 長時間作業して疲れ切ってしまうことが多い方
- 自分の作業量を把握して、計画的に進めたい方
- リモートワークやフリーランスで、自分でペースを管理する必要がある方

### PomoCare でできること

PomoCare はポモドーロテクニックの実践を手軽にサポートするアプリです。タイマーの管理だけでなく、作業の記録・振り返りまでをカバーします。

- タイマーが作業→休憩→作業のサイクルを **自動で管理**
- セッションごとに **ラベルやメモ** を付けて記録
- **統計グラフ** で日別・週別・月別の作業量を振り返り
- アラーム音やテーマなど、自分好みに **カスタマイズ**
- アカウント登録で **複数デバイスからデータ同期**

### アカウントなしでも使えます

PomoCare はログインしなくてもすべての基本機能を利用できます。データはブラウザのローカルストレージに保存されます。アカウントを作成すると、クラウドにデータが同期され、複数デバイスで利用できるようになります。

## 2. 画面の見かた

アプリは大きく3つの画面で構成されています。

### ヘッダー（共通）

```
 [ロゴ]          [統計] [設定] [ユーザー]
```

| ボタン | 説明 |
|--------|------|
| ロゴ | アプリのホーム（タイマー画面）に戻る |
| 統計 | 統計・グラフ画面を開く |
| 設定 | 設定画面を開く |
| ユーザー | ログイン、またはユーザーメニュー（ログアウトなど）を表示 |

### メイン画面（タイマー）

```
┌────────────────────────────┐
│         25:00              │  タイマー表示
│                            │
│     [▶ 開始]  [↺ リセット]  │  操作ボタン
│                            │
│  今日: 3回 (1時間15分)      │  セッションサマリー
│  今週: 12回 (5時間)         │
│                            │
│  [ラベル選択 ▼]             │  ラベル選択
│  [メモを入力...]            │  メモ入力欄
└────────────────────────────┘
```

## 3. タイマーを使う

### 基本操作

| 操作 | 方法 |
|------|------|
| **開始** | ▶ ボタンをタップ。タイマーが動き出し、フォーカスモードに切り替わります |
| **一時停止** | フォーカスモード中に ⏸ ボタンをタップ |
| **再開** | 一時停止中に ▶ ボタンをタップ |
| **リセット** | 一時停止中に表示されるリセットボタンをタップ（確認ダイアログが出ます） |
| **早期完了** | 作業中に「完了」ボタンをタップすると、途中でもセッションとして記録されます |

### 自動サイクル

タイマーが0になると自動で次のモードに切り替わります。

```
作業 → 短い休憩 → 作業 → 短い休憩 → ... → 作業 → 長い休憩 → 作業 → ...
```

長い休憩は、設定した回数（デフォルト: 4回）の作業セッション完了後に自動で発生します。

## 4. フォーカスモード（作業中の集中画面）

タイマーを開始すると、全画面の集中表示に切り替わります。余計な情報を排除し、タイマーだけに集中できます。

### 実行中に表示されるボタン

| ボタン | 動作 |
|--------|------|
| **停止** | タイマーを一時停止する |
| **完了** | セッションを途中で完了する（経過時間が記録されます） |

### 一時停止中に表示されるボタン

| ボタン | 動作 |
|--------|------|
| **再開** | タイマーを再開する |
| **リセット** | タイマーをリセットしてタイマー画面に戻る |

リセットや早期完了時は、誤操作防止のための確認ダイアログが表示されます。

---

## 5. 休憩モード

作業セッション完了後、自動的に休憩モードに切り替わります。

- **短い休憩**: 通常のセッション後（デフォルト: 5分）
- **長い休憩**: 設定回数のセッション完了後（デフォルト: 4回ごとに20分）

### 休憩中の操作

| 操作 | 説明 |
|------|------|
| **タイマー開始** | 休憩タイマーを開始する |
| **スキップ** | 休憩をスキップして次の作業セッションへ進む |

休憩中は、設定で登録したカスタムメッセージが表示されます（例:「水を飲みましょう」「ストレッチしましょう」など）。

## 6. ラベルでタスクを分類する

ラベルを使うと、何の作業をしたかを色分けして記録できます。

### ラベルの使い方

1. タイマー画面の **ラベル選択ドロップダウン** からラベルを選ぶ
2. タイマーを開始すると、選択中のラベルがセッションに自動的に記録される
3. ラベルなしでもセッションの記録は可能

### ラベルの新規作成

- タイマー画面のラベル選択から **「新規作成」** を選ぶと、その場でラベルを作成できます
- 設定画面のラベル管理からも作成可能です

### ラベルの管理（設定画面）

| 操作 | 方法 |
|------|------|
| **色を変更** | ラベル横のカラーアイコンをタップし、56色パレットまたはカスタムカラーから選択 |
| **名前を変更** | ラベル名をタップして直接編集 |
| **並び替え** | ラベルをドラッグ&ドロップで順序を変更（タッチ操作対応） |
| **削除** | 削除ボタンをタップ（確認ダイアログあり） |

---

## 7. セッションにメモを残す

作業内容の詳細をメモとして記録できます。

### メモの入力

1. タイマー画面でラベルを選択すると、メモ入力欄が表示されます
2. テキストを入力してからタイマーを開始すると、セッションと一緒に保存されます

### メモの後から編集

1. 統計画面で日付バーをタップ → セッション詳細モーダルを表示
2. 該当セッションのメモ欄を直接編集

---

## 8. 統計・グラフで振り返る

ヘッダーの **統計ボタン** をタップすると、作業の記録をグラフで確認できます。

### セッションサマリー（タイマー画面上部）

| 表示項目 | 内容 |
|----------|------|
| **今日** | 本日のセッション数と合計作業時間 |
| **今週** | 今週のセッション数、アクティブ日数（X/7）、合計時間 |

タップすると詳細なセッション一覧を表示します。

### 3つの表示期間

| ビュー | 表示内容 |
|--------|----------|
| **週間** | 過去7日間の日別グラフ |
| **月間** | 当月の日別グラフ |
| **年間** | 12ヶ月の月別グラフ |

**◀ ▶ ボタン** で前後の期間に移動できます。

### グラフの表示切替

| モード | 表示内容 |
|--------|----------|
| **セッション数** | 各日/月の完了セッション数 |
| **合計時間** | 各日/月の合計作業時間 |

### その他のグラフ機能

- **積み上げ棒グラフ**: 「全体表示」の場合、ラベルごとに色分けされた積み上げグラフで表示
- **ラベルフィルター**: 特定のラベルだけを表示するフィルター
- **グラフ高さの変更**: グラフ下部をドラッグして高さを調整（40px〜300px）
- **ホバーツールチップ**: グラフにカーソルを合わせるとセッション数と時間を表示

### ラベル別統計

グラフの下に、ラベルごとの統計が表示されます。

- セッション数
- 合計時間
- プログレスバーによる比率表示

### セッション詳細モーダル

グラフの日付バーをタップすると、その日のセッション一覧が表示されます。

| 操作 | 説明 |
|------|------|
| **ラベル変更** | セッションのラベルをインラインで変更 |
| **メモ編集** | メモを直接編集 |
| **セッション削除** | 個別セッションを削除（確認付き） |

## 9. セッションデータの管理（CSV）

### エクスポート（書き出し）

設定画面 → **「CSVエクスポート」** をタップすると、全セッションデータがCSVファイルとしてダウンロードされます。

### インポート（読み込み）

設定画面 → **「CSVインポート」** で、CSVファイルを読み込めます。

- ファイルをドラッグ&ドロップ、またはファイル選択で読み込み
- Shift-JIS / UTF-8 のエンコーディングを自動判別
- 同一日付のデータは重複排除されます
- CSVに含まれる未登録のラベルは自動作成されます

## 10. 設定を変更する

ヘッダーの **設定ボタン** をタップして設定画面を開きます。

### タイマー時間

| 項目 | 説明 | デフォルト |
|------|------|-----------|
| **作業時間** | プリセットから選択、またはカスタム値を入力 | 25分 |
| **短い休憩** | プリセットから選択、またはカスタム値を入力 | 5分 |
| **長い休憩** | ON/OFF切替、間隔（何セッションごとか）と時間を設定 | 4セッションごと / 20分 |

### プリセット編集

作業時間・休憩時間のクイック選択ボタンに表示される値を追加・削除できます。

### アラーム

| 項目 | 説明 |
|------|------|
| **アラーム音** | 8種類から選択（Bell, Digital, Chime, Kitchen, Classic, Gentle, Soft, なし） |
| **リピート回数** | アラームの繰り返し回数（1〜5回） |
| **プレビュー** | 選択中のアラーム音を試聴 |

#### アラーム音の特徴

| 音名 | 特徴 |
|------|------|
| **Bell** | ハンドベル風の澄んだ音 |
| **Digital** | 目覚まし時計風のデジタル音 |
| **Chime** | 3音のアルペジオ（ド-ミ-ソ） |
| **Kitchen** | キッチンタイマー風の音 |
| **Classic** | クラシックなアラーム音 |
| **Gentle** | 穏やかな通知音 |
| **Soft** | 柔らかいソフトな音 |
| **なし** | サイレント（音なし） |

### 外観

| 項目 | 説明 |
|------|------|
| **テーマ** | ライトモード / ダークモードを切り替え |
| **言語** | 7言語から選択 |

### カスタムメッセージ

休憩中に画面に表示するメッセージを自由に設定できます。

例: 「水を飲みましょう」「目を休めましょう」「ストレッチタイム！」

### データ管理

| 操作 | 説明 |
|------|------|
| **データリセット** | すべてのセッションデータ・設定を消去します（確認ダイアログあり） |
| **アカウント削除** | ログイン中のアカウントを完全に削除します（確認ダイアログあり） |

## 11. アカウントとデータ同期

### ログイン方法

ヘッダー右端のユーザーアイコンをタップすると、ログインモーダルが開きます。

| 方法 | 説明 |
|------|------|
| **Googleログイン** | Googleアカウントでワンタップログイン |
| **メール/パスワード** | メールアドレスとパスワードで新規登録またはログイン |

### アカウント作成の流れ

1. メールアドレスとパスワードを入力して **サインアップ**
2. 確認メールが届くので、メール内のリンクをクリック
3. メール認証が完了すると、すべての機能が利用可能に

### パスワードを忘れた場合

1. ログインモーダルで **「パスワードを忘れた方」** をタップ
2. メールアドレスを入力して送信
3. 届いたメールのリンクからパスワードを再設定

### データの保存先

| 状態 | 保存先 | 特徴 |
|------|--------|------|
| **未ログイン** | ブラウザのローカルストレージ | そのブラウザでのみ利用可能 |
| **ログイン済み** | クラウド（Firestore） | 複数デバイスで同期 |

### 初回ログイン時のデータ移行

未ログインの状態で記録したセッションデータは、初めてログインした際に自動的にクラウドへ移行されます。

## 12. アプリのインストール（PWA）

PomoCare はブラウザからインストールして、ネイティブアプリのように使えます。

### インストール方法

1. ブラウザでPomoCareを開く
2. 画面下部に表示される **インストールバナー** をタップ
3. ブラウザの案内に従ってインストール

### インストール後の特徴

- ホーム画面/デスクトップからアプリとして起動
- ブラウザのアドレスバーなしのフルスクリーン表示
- オフラインでも動作（インターネット接続なしで利用可能）
- アプリの更新は自動で適用

## 15. よくある質問（FAQ）

### Q. ブラウザを閉じてもデータは消えませんか？

**A.** 未ログインの場合、データはブラウザのローカルストレージに保存されます。ブラウザのデータを消去しない限り残ります。ログイン済みの場合はクラウドに保存されるため、別のデバイスからもアクセスできます。

### Q. 作業時間を25分以外に変えられますか？

**A.** はい。設定画面の「作業時間」からプリセットを選ぶか、カスタム値を入力できます。休憩時間も同様にカスタマイズ可能です。

### Q. 長い休憩を使わないことはできますか？

**A.** はい。設定画面で長い休憩をOFFにできます。

### Q. セッションを途中で終わらせた場合はどうなりますか？

**A.** 「早期完了」を選ぶと、実際に作業した時間がそのまま記録されます。例えば25分のタイマーで15分経過後に完了した場合、15分のセッションとして記録されます。

### Q. 他のアプリで記録したデータを移行できますか？

**A.** CSV形式のデータであればインポート機能で読み込めます。設定画面の「CSVインポート」から、ファイルをドラッグ&ドロップまたは選択して読み込んでください。

### Q. オフラインでも使えますか？

**A.** はい。アプリをインストール（PWA）すれば、オフラインでもタイマーや記録機能を利用できます。ただし、クラウド同期はインターネット接続時に行われます。

### Q. 対応しているブラウザは？

**A.** Chrome, Edge, Safari, Firefox など主要なモダンブラウザに対応しています。PWAのインストール機能はChromeとEdgeで最も安定して動作します。
